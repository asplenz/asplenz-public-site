# Asplenz Horizon Website

Institutional evidence infrastructure website built with Next.js.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Content**: Markdown with gray-matter

## Structure

- `/app` - Next.js application routes
- `/components` - React components
- `/lib` - Utilities (markdown parsing, navigation)
- `/content` - Bilingual markdown content

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Languages

The site is fully bilingual (EN/FR) with routes:
- `/en/*` - English content
- `/fr/*` - French content

## Design Principles

- Radical minimalism
- Black and white only
- Institutional haute couture aesthetic
- Optimized reading experience
- No em-dashes, use colons or periods
